# Chesapeake Bay Wetland Dynamics SDSS

**Web-based Spatial Decision Support System** for the paper:  
**"Deep Learning Mapping of Wetland Dynamics: A User-Friendly Web-Based Decision Support Tool for Chesapeake Bay"** (Mahmoudnia, 2026)

## What is included in this repository
- Complete Flask + Leaflet dashboard (local prototype)
- Pre-computed static wetland change maps and metrics used by the dashboard

## How to run the dashboard locally
1. Clone or download this repository
2. `cd Chesapeake-Wetland-SDSS`
3. `pip install -r requirements.txt`
4. `python app.py`
5. Open http://127.0.0.1:5000 in your browser

Tested with Python 3.10–3.12.
